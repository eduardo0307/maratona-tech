Quiz de Segurança Online

Teste seus conhecimentos sobre segurança online e como proteger suas informações pessoais na internet.

1. O que é um phishing?

  • A) Uma tentativa de obter informações confidenciais enganando o usuário
  • B) Um tipo de software antivírus
  • C) Um protocolo de rede

2. O que significa usar uma senha forte?

  • A) Uma senha com letras, números e caracteres especiais
  • B) Uma senha com apenas letras
  • C) Uma senha de 4 dígitos

3. O que é autenticação de dois fatores (2FA)?

  • A) Um método de segurança que requer dois tipos diferentes de prova para acessar uma conta
  • B) Um software para criptografia de e-mails
  • C) Um tipo de firewall

4. O que é um firewall?

  • A) Um sistema de segurança que controla o tráfego de rede
  • B) Um tipo de software para gestão de senhas
  • C) Uma técnica de criptografia

5. Qual é a principal função de um antivírus?

  • A) Detectar e remover vírus e malware do computador
  • B) Monitorar o tráfego de rede
  • C) Gerenciar e-mails

6. O que é um ataque de força bruta?

  • A) Um método de tentativa e erro para quebrar senhas
  • B) Um tipo de malware
  • C) Uma técnica de engenharia social

7. O que é criptografia?

  • A) O processo de codificar informações para proteger sua confidencialidade
  • B) Um tipo de firewall
  • C) Um programa para backup de dados

8. O que é engenharia social?

  • A) Manipulação psicológica para obter informações confidenciais
  • B) Um tipo de software antivírus
  • C) Um protocolo de rede

9. O que são dados pessoais?

  • A) Informações que podem identificar uma pessoa
  • B) Arquivos de sistema do computador
  • C) Programas de software

10. Por que é importante atualizar o software regularmente?

  • A) Para corrigir vulnerabilidades de segurança e melhorar o desempenho
  • B) Para aumentar o consumo de memória
  • C) Para alterar o formato dos arquivos
