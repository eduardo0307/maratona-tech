Conceitos Básicos de Internet

Responda às perguntas abaixo para testar seus conhecimentos sobre conceitos básicos de internet.

1. O que é a internet?

  • A) Uma rede de computadores interconectados globalmente
  • B) Um software para editar textos
  • C) Um tipo de vírus de computador

2. Qual é o propósito do protocolo HTTP?

  • A) Transferir arquivos entre computadores
  • B) Garantir a segurança das informações
  • C) Facilitar a comunicação entre navegadores e servidores web

3. O que significa a sigla URL?

  • A) Uniform Resource Locator
  • B) Universal Resource Language
  • C) Uniform Resource Language

4. O que é um IP?

  • A) Um tipo de software de segurança
  • B) Um endereço único atribuído a cada dispositivo na internet
  • C) Um protocolo de comunicação

5. O que faz um navegador web?

  • A) Armazena dados em cache
  • B) Permite a visualização e interação com sites web
  • C) Gerencia redes de computadores

6. Qual é a função de um servidor DNS?

  • A) Traduzir URLs em endereços IP
  • B) Enviar e-mails
  • C) Proteger a rede contra ataques

7. O que é um protocolo de rede?

  • A) Um conjunto de regras para comunicação entre dispositivos
  • B) Um tipo de vírus de computador
  • C) Um software antivírus

8. O que é um website?

  • A) Uma coleção de páginas da web acessíveis na internet
  • B) Um programa de e-mail
  • C) Um tipo de malware

9. Qual é a função do protocolo TCP?

  • A) Garantir a entrega correta e completa de dados entre dispositivos
  • B) Gerenciar as conexões de e-mail
  • C) Realizar backups de dados

10. O que é um firewall?

  • A) Um sistema de segurança que controla o tráfego de rede
  • B) Um tipo de software para edição de vídeos
  • C) Um serviço de armazenamento na nuvem
